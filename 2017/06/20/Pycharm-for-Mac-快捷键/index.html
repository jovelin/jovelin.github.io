Pycharm for Mac 快捷键

ctrl + r 运行

ctrl + d 调试

ctrl + shift + r 从编辑器运行上下文环境配置

ctrl + shift + d 从编辑器调试上下文环境配置

ctrl + opt + r 弹出 Run 的可选菜单

ctrl + opt + d 弹出 Debug 的可选菜单

fn + f8 单步调试，跳过函数

fn + f7 单步调试，会进入函数内部

cmd + 1 展开收缩左边文件栏

cmd + / 注释/取消注释一行

fn + ctrl + space 基本代码补全（补全任何类、方法、
变量）

cmd + shift + -/+ 收缩/展开代码块

cmd + opt + l 格式化代码

fn + cmd + f2 退出运行

cmd + shift + 上/下 移动代码块

cmd + z 撤回

cmd + shift + z 取消撤回

cmd + opt + r 跳到下一个断点

fn + cmd + shift + f8 查看所有断点

ctrl + j 查看函数文档注释
